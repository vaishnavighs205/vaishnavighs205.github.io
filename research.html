About the Lab

The Bolser Lab studies how the nervous system produces and regulates airway defensive behaviors like coughing and breathing. The lab uses multi-electrode array technology to record brainstem neuron activity and builds computational models of the brainstem cough network to understand how neural circuits coordinate these behaviors and how drugs modulate them. Current projects include modeling upper airway, swallowing, and breathing behaviors, and studying how the brainstem respiratory network responds to different stimuli.

Working on NeuronetExperimenter, a neural network simulation platform built to model realistic networks of mammalian brainstem neurons [1]. The software simulates how neurons in the brainstem coordinate airway protective behaviors, with the longer-term goal of running large-scale simulations on UF's HiPerGator supercomputing cluster.

  • Developing and testing features for NeuronetExperimenter using Qt Creator and Qt C++, improving the GUI for simulation control and parameter input [1]
  • Writing Python scripts to generate advanced visualizations of simulated neuron activity, translating raw simulation outputs into clear, analysis-ready figures [1]
  • Supporting the lab's computational modeling work on brainstem respiratory circuits involved in cough, swallowing, and breathing behavior
  • Conducting tool functionality testing to identify bugs and improve usability ahead of future HiPerGator deployment [1]
  • Contributing to efforts that connect neural simulation outputs to observed physiological behaviors in mammalian models
